DEPLOYMENT WORKFLOW
================================================================================

1. GIT PUSH
   git add -A
   git commit -m "Add Render deployment configuration"
   git push origin main

2. CREATE BACKEND SERVICE
   - Render Dashboard → New Web Service
   - Configure FastAPI service
   - Set environment variables
   - Deploy (~2 minutes)

3. CREATE FRONTEND SERVICE
   - Render Dashboard → New Web Service
   - Configure Streamlit service
   - Set BACKEND_URL from Step 2
   - Deploy (~2 minutes)

4. TEST
   - Health: backend.onrender.com/health
   - Frontend: frontend.onrender.com
   - Workflow: Enter YouTube URL → Process

5. SETUP NOTION
   - Create integration at notion.so/my-integrations
   - Update NOTION_API_KEY in backend
   - Grant integration access in Notion
   - Test full workflow

TOTAL TIME: ~20-25 minutes
